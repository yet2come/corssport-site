# 現行実装の補足メモ

この文書は、`docs/booking-system-design.md` にまだ反映しきれていない、現在の実装上の差分と運用メモをまとめたものです。

## 1. 施設名と表示文言

- `Event Space` は現在 `Event Hall / イベントホール` 表記で運用している
- `Meeting Room` の定員表示は `最大10名`
- `Solo Booth` の定員表示は `1名x5室`
- 予約ページには予約対象3施設に加えて、`Rent-a-car`、`Consultation`、`Study Visit` のカードがある

## 2. Solo Booth の実装仕様

- `Solo Booth` は単一カレンダーではなく、`GCAL_ID_SOLO_BOOTH_1` 〜 `GCAL_ID_SOLO_BOOTH_5` の 5 カレンダー構成
- 空き状況 API は 5 カレンダーを集計し、時間帯ごとの残室数を返す
- 予約時は空いている 1 室を自動割当する
- ユーザーは複数室を同時予約しない前提
- 営業時間は `09:00 - 18:00`

## 3. 予約フォームの現行仕様

- 時間帯は連続した複数スロットを選択できる
- ヒント文言は `連続した時間帯を複数選択できます`
- 入力ラベルは日本語化済み
- `利用目的` の項目名は `利用目的・連絡事項等`
- `メールアドレス` の項目名は `メールアドレス（確認メール送付先）`
- `Event Hall` 選択時のみ `レイアウト変更 +¥3,300` オプションが表示される

## 4. 料金表示の現行仕様

予約フォームには `ビジター料金` と `メンバー料金` の両方を表示する。

### Event Hall

- ビジター料金: 1時間 `¥8,800`
- ビジター料金: 18:00以降 1時間 `¥13,200`
- メンバー料金: 1時間 `¥6,600`
- メンバー料金: 18:00以降 1時間 `¥9,900`
- `レイアウト変更` はビジター・メンバーともに `+¥3,300`

### Meeting Room

- ビジター料金: 1時間 `¥2,200`
- ビジター料金: 18:00以降 1時間 `¥4,400`
- メンバー料金: 1時間 `¥1,100`
- メンバー料金: 18:00以降 1時間 `¥3,300`

### Solo Booth

- ビジター料金: 2時間まで `¥1,200`
- ビジター料金: 以降 1時間ごとに `¥600`
- ビジター料金: 上限 `¥2,500`
- メンバー料金: 2時間まで `¥600`
- メンバー料金: 以降 1時間ごとに `¥300`
- メンバー料金: 上限 `¥1,000`
- Solo Booth は `18:00` まで

## 5. 予約・キャンセル導線

- `SITE_URL` は現在 `https://crossport.cc` 前提
- 予約確認メール内のキャンセルリンクは `https://crossport.cc/cancel.html#...`
- `book.html` と `cancel.html` は `noindex,follow`
- `admin-manual.html` は `noindex,nofollow`

## 6. SEO と公開設定

- 本番ドメインは `crossport.cc`
- トップページには `canonical`、OG、Twitter Card、JSON-LD を設定済み
- `robots.txt` と `sitemap.xml` を配置済み
- `sitemap.xml` にはトップページ `/` のみを掲載している

## 7. トップページの現行 UI 反映事項

- ヘッダー下ティッカーは `壱岐なう` の iframe 埋め込み
- ACCESS セクションの見出し直下にも `壱岐なう` の iframe 埋め込みを入れている
- ABOUT セクションの4項目は以下
  - 始める拠点
  - 人が交わる場所
  - 島で動くベース
  - 小さく試す
